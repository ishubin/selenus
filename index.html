Selenus Introduction

Selenus is a web testing framework based on Selenium and Oculus Experior. It offers a comprehensive support for designing tests and pages and makes use of Selenium and web testing a breeze.

  • Page models Selenus offers rich support for pages and page objects. This helps to keep your test code clean and flexible.
  • Verification For all page objects there are two types of assertions implemented. You can use assertThat or verifyThat in many places for different page objects.
  • Nice reports Every action of Selenus page objects is being reported using Oculus Experior reports. It is well structured and easy to read. Also it is very easy to attach sccreenshots to your reports.
  • Integration with Frontend As Selenus is based on Oculus Experior it also has integration with Oculus Frontend. For this you also need Oculus Frontend webapp.
